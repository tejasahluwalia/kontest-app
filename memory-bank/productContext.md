# Product Context: Kontest App

## Project Overview
Kontest App is a platform for creating and managing contests. It allows organizations to host contests and participants to join and compete in these contests.

## Key Components
- **Organizations**: Groups that can host contests
- **Contests**: Events created by organizations
- **Hosts**: Users who manage organizations and create contests
- **Participants**: Users who join and participate in contests

## Project Organization
- **Client**: Frontend React application
- **Server**: Backend API server
- **Database**: PostgreSQL database managed through Drizzle ORM

## Technical Standards
- TypeScript for type safety
- React for frontend
- Express for backend API
- Drizzle ORM for database operations
- Docker for containerization

## Current Implementation Status
Building out the host functionality to manage organizations and contests.
